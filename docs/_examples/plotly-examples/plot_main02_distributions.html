Plot DistributionsÔÉÅ

This example shows how to plot the probability distributions for each of the components of the confusion matrix. Note that data is created artificially. If necessary, it is possible to limit the axis to the range [0, 1] to keep only those true probability values.

 29 # Helper method

 31 # This method is implemented in pySML.
 32 def _tp_fp_tn_fn_distributions(y, y_pred, y_prob):
 33     """This function returns probabilities for each of the confusion
 34     matrix elements (tp, tn, fp, fn).
 35
 36     Parameters
 37     ----------
 38     y : array-like
 39       The real categories
 40
 41     y_pred : array-like
 42       The predicted categories
 43
 44     y_prob: array-like
 45       The predict probabilities
 46
 47     Returns
 48     -------
 49     tp_probs, tn_probs, fp_probs, fn_probs
 50     """
 51     # Tags.
 52     tp_idx = (y_pred == 1) & (y == 1)
 53     tn_idx = (y_pred == 0) & (y == 0)
 54     fp_idx = (y_pred == 1) & (y == 0)
 55     fn_idx = (y_pred == 0) & (y == 1)
 56     # Show information.
 57     tp_probs = y_prob[tp_idx]
 58     tn_probs = y_prob[tn_idx]
 59     fp_probs = y_prob[fp_idx]
 60     fn_probs = y_prob[fn_idx]
 61     # Return
 62     return tp_probs, tn_probs, fp_probs, fn_probs
